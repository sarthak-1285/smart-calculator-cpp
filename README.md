# Smart Calculator (C++)

A menu-driven calculator built using C++ that performs basic arithmetic operations through an interactive console interface.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Repeated calculations using loops
- Division by zero handling
- Invalid choice validation
- Exit option

---

## Concepts Used

This project demonstrates the following C++ concepts:

- Variables and Data Types
- Input and Output (`cin`, `cout`)
- Arithmetic Operators
- Conditional Statements (`if-else`)
- Loops (`do-while`)
- Switch Statements
- Type Casting
- Basic Error Handling

## How to Run

### Compile

```bash
g++ main.cpp -o calculator
```

### Run (Linux/macOS)

```bash
./calculator
```

### Run (Windows)

```bash
calculator.exe
```
